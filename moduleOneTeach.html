Important Java Concepts:

  • Java is Object oriented: A programming language where the user can create and manipulate objects alongside its attributes. For example, an object created could be a car, and its attributes would be the colour, model, and license plate.
  • Variables: In java, variables are used to store data values. They can be used to store the attributes for objects, or just general data such as numbers, words, and characters. There are many different types of variables.
  • Data Types: Varaibles in java can be defined by boolean (store true or false), int (store whole numbers), double (store fractional numbers), string (store words and sentences).
